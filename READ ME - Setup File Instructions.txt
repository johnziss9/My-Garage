1) Use command "Install-Package System.Data.SQLite.Core -Version 1.0.112" to install SQLite.Core through Package Manager Console.
2) Download sqlite.interop.dll and add it to the Application Folder along with all the other Assemblies and the Application output.
3) Right click on the project in Solution Explorer (Not the setup project), click on Properties, then click on the Build tab and untick Prefer 32-bit.
